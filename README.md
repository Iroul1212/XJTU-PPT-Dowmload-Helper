## 使用教程
### 第一步：准备工作（如果你还没做的话）
安装插件：确保你的浏览器（Chrome, Edge, Firefox 等）已经安装了 Tampermonkey (篡改猴) 扩展插件。
- Edge 商店链接：https://microsoftedge.microsoft.com/addons/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/iikmkjmpaadaobahmlepeloendndfphd
  
### 第二步：创建新脚本
1. 点击浏览器右上角的 Tampermonkey 图标。
2. 选择 “添加新脚本” (Create a new script)。
3. 你会看到一个编辑器，里面已经有一些默认的代码（元数据块）。

### 第三步：导入脚本
1. 全选默认代码并删除
2. 将本脚本内容全选复制进去

### 第四步：保存并使用
1. 点击编辑器左上角的 “文件” -> “保存”（或者直接按 Ctrl + S）。
2. 打开思源学堂，现在可以随便下载了
